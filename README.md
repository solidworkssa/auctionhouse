# AuctionHouse

Simple auction contract with bidding.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/AuctionHouse.sol`
- Stacks: `contracts/stacks/contracts/auctionhouse.clar`

## Setup
```bash
pnpm install
pnpm dev
```
